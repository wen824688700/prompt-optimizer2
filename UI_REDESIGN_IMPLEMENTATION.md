## 🎨 设计特点

### 配色方案
- **主色调**: 渐变紫色 (#667eea → #764ba2)
- **辅助色**: 青色 (#06b6d4)、粉色 (#ec4899)
- **背景**: 深色渐变 (indigo-950 → purple-900 → slate-900)
- **页脚**: 纯黑 (#1a1a1a)

### 动画效果
- **首屏**: fade-in-up 动画，依次延迟出现
- **卡片**: 悬停时上移 + 缩放
- **FAQ**: 平滑展开/收起动画
- **背景**: 动态光晕脉冲效果

### 响应式设计
- **桌面端**: 3列网格布局
- **移动端**: 单列布局
- **横向滚动**: 触摸友好的卡片滚动
